# 👋 Comece aqui — Uni AVP Fase 1

1. Schema do banco completo (8 tabelas, RLS, função de progressão)

## 📖 Por onde começar
1. Leia `README.md` e siga o setup (15 min)
2. Crie conta Supabase + rode o SQL
3. Configure `.env.local`
4. `npm install && npm run dev`
5. Cadastre-se → promova-se a admin
